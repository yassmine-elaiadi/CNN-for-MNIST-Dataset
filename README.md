# CNN-for-MNIST-Dataset 
##Goal of the project 
The aim of this project was to implement and analyze a convolutional neural network (CNN) for
handwritten digit recognition using the MNIST dataset. In addition to reproducting the basic
CNN approach introduced in the lecture , several experiments were conducted in order to inves
tigate how different training parameters and architectural decisions influence the classification
performance . In accordance with the project requirement, the focus of this e-portfolio is not
only on the final accuracy of the model but also on the development process, the intermediate
results, the decisions taken during experimentation, and the insights gained during the work
process.
